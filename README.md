# smilify - Sistema de Votação de Satisfação do Cliente

Este projeto consiste em um sistema de votação para avaliar o atendimento aos clientes, com 5 opções de rostinhos para representar diferentes níveis de satisfação. A aplicação foi desenvolvida utilizando Vue 3 para o frontend e faz uso do Axios para consumir uma API que registra os votos dos clientes.

## Funcionalidades

- Apresenta uma página simples para os clientes escolherem uma entre as 5 opções de rostinhos que representam a satisfação com o atendimento.
- Registra o voto do cliente consumindo uma API para armazenar as respostas.

## Tecnologias Utilizadas

- Vue 3: Utilizado para a construção do frontend da aplicação.
- Axios: Biblioteca para fazer requisições HTTP para a API de registro de votos.

## Instalação e Uso

1. Certifique-se de ter o Node.js instalado na sua máquina.
2. Clone este repositório.
3. Navegue até o diretório do projeto e instale as dependências:

    ```bash
    npm install
    ```

4. Após a instalação, inicie a aplicação:

    ```bash
    npm run serve
    ```

5. Acesse a aplicação localmente em seu navegador:

    ```
    http://localhost:8080
    ```

## Configuração da API

Certifique-se de configurar a URL da API corretamente no arquivo de configuração ou variáveis de ambiente para que o frontend consiga enviar os votos dos clientes para a API de registro.
